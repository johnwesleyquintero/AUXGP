# Cold Email Sequence

## Frequency Plan

| Week       | Cadence                      |
| ---------- | ---------------------------- |
| **Week 1** | 1 email per lead             |
| **Week 2** | 2 follow-ups during the week |

---

# Sequence Overview

| Step        | Timing           | Goal                              |
| ----------- | ---------------- | --------------------------------- |
| **Email 1** | Initial Outreach | Introduce AuxGP value proposition |
| **Email 2** | Follow-Up #1     | Reinforce staffing pain points    |
| **Email 3** | Follow-Up #2     | Social proof / industry demand    |
| **Email 4** | Breakup Email #1 | Soft close with options           |
| **Email 5** | Breakup Email #2 | Final touch / leave door open     |

# Notes for Optimization

## Recommended Improvements

| Area            | Suggestion                               |
| --------------- | ---------------------------------------- |
| Personalization | Add company name, portfolio size, region |
| CTA             | Use one CTA only per email               |
| Deliverability  | Keep under 120 words where possible      |
| A/B Testing     | Test curiosity vs direct subject lines   |
| Timing          | Tue–Thu mornings EST                     |

---

# Strategic View

This sequence is solid because it follows an old-school truth:

**Respectful persistence beats aggressive selling.**

You’re not spamming. You’re professionally staying visible until timing aligns.
